# Мой первый заказной проект

Это мой первый заказной веб-проект: официальный сайт блогера с поддержкой нескольких языков. Создан Никитой Дюковым.

## О проекте
- Веб-сайт блогера с большим количеством подписчиков.
- Поддержка нескольких языков (русский, английский, испанский, китайский).
- Создан: 25.12.2025.
- Время работы: ~8 часов.

## Использование
Откройте index.html в браузере, чтобы выбрать язык и перейти на нужную версию сайта.
